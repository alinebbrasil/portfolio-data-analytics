# Portfólio de Dados

Este repositório reúne projetos de dados desenvolvidos com foco em aplicação prática, abrangendo diferentes ferramentas e etapas do ciclo de dados, desde a preparação até a visualização e geração de insights.

## Objetivo

O objetivo deste portfólio é demonstrar habilidades em análise de dados, engenharia de dados e visualização, utilizando ferramentas amplamente utilizadas no mercado.

## Ferramentas e Tecnologias

- Python (Pandas, manipulação e análise de dados)
- SQL (consultas analíticas e exploração de dados)
- Power BI (visualização e construção de dashboards)
- Jupyter Notebook

Cada projeto está organizado em uma pasta própria, contendo:

- Consultas SQL
- Arquivos de visualização

## Observações

Este portfólio está em constante evolução, com a inclusão de novos projetos e tecnologias ao longo do tempo.
